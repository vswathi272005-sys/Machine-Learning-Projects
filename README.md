# Customer Churn Prediction
## Description
This project predicts whether a customer is likely to leave a company or continue using its services using Machine Learning.
## Technologies Used
- Python
- Pandas
- Scikit-learn
- Logistic Regression
## Features
- Data Preprocessing
- Customer Churn Prediction
- Model Training
- Prediction
## How to Run
```bash
python app.py
```
## Output
The model predicts whether the customer will:
- Stay
- Churn
